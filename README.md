# college-app-project

Salem College Super App
    A three page informational website dedicated to salem college. Includes a homepage, a Campus Life page, and an About Us page, all of which provide information about salem college meant for student use.
    
    Homepage (index.html) provides a general overview and introduction to salem college, with current news/events and quick statistics related to the college. Campus Life (page2.html) provides information related to campus activities, dining/recreational facilities, and housing information. About Us (page3.html) provides more detailed information about the college, such as the mission/vision statements, and extended list of statistics, Salem's accreditation, core values, and a leadership directory for students to have quick access to Salem College leadership contact information


Technologies used
    -   Visual Studio
    -   Live Server
    -   Lighthouse (Accessability check)


Instructions for setup and deployment
    Visit [*website](https://nfull.github.io/college-app-project/) OR 
    -   Clone repository onto your device
    -   Open with Visual Studio
    -   Use Live Server to preview website


Overview of features and how to use them
    Each page has a header and a footer. The header includes three buttons to each of the pages. The footer also includes buttons to each of the pages, as well as buttons that lead to external sites related to the button name (ex: Financial Aid leads to external Salem College website for Financial Aid).

    The Homepage (index.html) includes a "Latest News and Events" section, where there is a preview of two articles. Each button for each preview leads to an external site that has the full article to read.

    Campus Life (page2.html) begins with three cards that give an overview of salem college facilities, those of which are housing, dining, and recreation. Each card has a link at the bottom that takes you to an external Salem College site related to the facility. At the bottom of the page, there is three buttons labeled "Events", "Pictures", and "Calendar". When you click on each button, a different link will show right below the buttons, leading to an external site related to the button.

    About Us (page3.html) has a leadership directory located at the bottom of the page. This directory includes a button, which toggles the visibility of the leadership cards, and a search bar, which allows someone to search for a specific role or person.
